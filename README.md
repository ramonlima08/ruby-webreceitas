## Projeto Web Receitas

Prejeto desenvolvido em Ruby on Rails para fixar conteúdo

### Sobre o site

Site de receitas

- Cadastro, Edição, Listagem, Visualização e Exclusão de Receita
- Validações nos campos das Receitas
- Feedbacks aos usuários
- Criação e Listagem de comentários
- Validações na criação dos comentários
